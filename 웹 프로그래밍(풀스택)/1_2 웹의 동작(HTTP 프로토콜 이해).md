# 인터넷 (네트웍 통신)의 이해

 * ### 인터넷 != WWW(**W**orld **W**Ide **W**eb)  
 WWW가 인터넷의 전부는 아니다.

| Name | Protocol | Port | Features |
|---|---|---|---|
| WWW | HTTP | 80 | 웹서비스 |
| Email | SMTP / POP3 / IMAP | 25 / 110 / 114 | 이메일 서비스 |
| FTP | FTP | 21 | 파일 전송 서비스 |
| DNS | TCP / UDP | 53 | 네임서비스 |
| NEWS | NNTP | 119 | 인터넷 뉴스 서비스 |
