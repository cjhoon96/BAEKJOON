```ABAP
DATA: gs_where    TYPE c LENGTH 72,
      gv_carrname TYPE scarr-carrname,
      gv_carrid   TYPE scarr-carrid VALUE 'AC'.

CONCATENATE 'CARRID = ''' gv_carrid '''' INTO gs_where.

SELECT SINGLE carrname
  INTO gv_carrname
  FROM scarr
  WHERE (gs_where).
WRITE:/ gv_carrname,
      / gs_where.

*Air Canada
*CARRID = 'AC'
```

와 같이 WHERE  조건문을 동적으로 쓸 수 있다.
