  1. CSS

    2. 종속형 시트 또는 캐스케이딩 스타일 시트(Cascading Style Sheets, CSS)는 마크업 언어가 실제 표시되는 방법을 기술하는 언어로, HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있다. W3C의 표준이며, 레이아웃과 스타일을 정의할 때의 자유도가 높다.

    마크업 언어(ex: HTML)가 웹사이트의 몸체를 담당한다면 CSS는 옷과 액세서리처럼 꾸미는 역할을 담당한다고 할 수 있다. 즉, HTML 구조는 그대로 두고 CSS 파일만 변경해도 전혀 다른 웹사이트처럼 꾸밀 수 있다.

    현재 개발 중인 CSS3의 경우 그림자 효과, 그라데이션, 변형 등 그래픽 편집 프로그램으로 제작한 이미지를 대체할 수 있는 기능이 추가되었다. 또한 다양한 애니메이션 기능이 추가되어 어도비 플래시를 어느 정도 대체하고 있다.

    Selector & Declaration & Property

    1. <style> </style>

      2. 해당 태그의 사이의 문법을 css로 하겠다는 것을 알려주는 태그로 헤드에 집어 넣는다
      모든 a 태그의 글자에 대해 빨간색으로 표기를 하겠다는 코드 html의 폰트 태그와 달리 a태그가 달린 모든 글자들의 색을 바꿀 수 있는 큰 효과가 있다.
      중복된 코드를 단 하나의 코드로 대체할 수 있다.
      O{}를 Selector(선택자)라고 하며 어느 부분에 효과를 줄지 지정해 준다. {} 안쪽의 효과는 Declaration(선언)이라고 한다. Declaration의 color 이나 test-decoration 을 Property 라고 한며 red나 underline, ex)
      <style> a{color:red;} </style>

    2. style="" 속성

      3. html 태그 안에 html의 속성 style="" 을 통해 ""안에 css의 효과를 집어넣는다.
      이 경우 태그 안쪽에 속성을 통해 효과를 주기 때문에 Selector(선택자)가 필요하지 않다.

    Declarations(Properties:Values;)

    1. color:'(색상)'

      2. 텍스트의 색을 바꿔주는 property로 value로 원하는 색상을 영어 또는 rgb(-,-,-) 을 통해 원하는 색상을 집어넣을수 있다.

    2. text-decoration:---

      3. 텍스트를 꾸며주는 property 로 value를 통헤 none으로 기존에 있던 설정을 없애줄 수 있으며 underline으로 밑줄을 만들어 주는 등의 효과를 줄 수 있다.

    3. font-size:--px

      4. 텍스트의 크기를 바꿔주는 property로 value값으로 숫자px를 줘 픽셀 값으로 크기를 정한다.

    4. text-align:---

      5. 텍스트의 배치를 바궈주는 property로 대표적으로 center가 있다.



      ex)
      <a href = "WEB_2(CSS).html"style = "color: red; text-decoration: underline; font-size:45px; text-align:center"> css </a> =>
